## 🎯 Overview

Food-Bridge is a web-based platform designed to bridge the gap between food surplus and food scarcity. Our mission is to:

- **Reduce Food Waste**: Connect restaurants, supermarkets, and individuals with surplus food to those in need
- **Build Community**: Foster connections between donors and recipients
- **Promote Sustainability**: Make sustainable food sharing accessible and convenient
- **Empower Users**: Provide an intuitive platform for giving and receiving food

---

## ✨ Features

- 🔐 **User Authentication**: Secure registration and login system for donors and recipients
- 📍 **Location-Based Matching**: Find nearby food donors or recipients based on location
- 🍲 **Food Listings**: Create detailed posts about available food with descriptions and images
- 💬 **Real-Time Communication**: Chat or messaging system between donors and recipients
- ⭐ **Rating & Reviews**: Build trust through community ratings and feedback
- 🔔 **Notifications**: Get instant updates on new donations and messages
- 📱 **Responsive Design**: Fully responsive interface for desktop and mobile devices
- 🎨 **Intuitive UI**: Clean and user-friendly interface for easy navigation

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Git** - [Download Git](https://git-scm.com/)
- **Node.js** (optional, if using a development server) - [Download Node.js](https://nodejs.org/)
- **Modern Web Browser** - Chrome, Firefox, Safari, or Edge

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/f25610059-pixel/Food-Bridge.git
   cd Food-Bridge
   ```

2. **Open the project:**
   - For a simple web browser view, open `index.html` directly in your browser
   - Or use a local development server (see Running the Project below)

### Running the Project

#### Option 1: Simple Browser Approach
1. Navigate to the project directory
2. Double-click `index.html` to open it in your default browser

#### Option 2: Using a Local Development Server

**With Python:**
```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

**With Node.js (http-server):**
```bash
npm install -g http-server
http-server
```

**With Node.js (Express):**
```bash
npm install
npm start
```

Then open your browser and navigate to `http://localhost:8000`

## 💡 Usage

### For Donors:
1. **Sign Up** as a donor with your information
2. **List Food**: Create a post with available food details
3. **Add Photos**: Upload images of the food
4. **Set Location**: Mark your donation location
5. **Connect**: Communicate with interested recipients
6. **Confirm Pickup**: Arrange delivery or pickup times

### For Recipients:
1. **Create Account** as a recipient
2. **Browse Listings**: View available food donations nearby
3. **Search & Filter**: Find food by category and location
4. **Request Food**: Send requests to donors
5. **Communicate**: Chat with donors to arrange details
6. **Collect/Receive**: Complete the donation transaction
